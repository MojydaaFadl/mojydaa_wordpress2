== Installation ==

= Minimum Requirements =
    To ensure smooth operation, WPelemento Importer requires the following:
        ● PHP Version: 5.4 or greater
        ● MySQL Version: 5.0 or greater
    
= Recommended Hosting Specifications =
    For optimal performance, we recommend your hosting environment supports:
        ● PHP Version: 7.0 or greater
        ● MySQL Version: 5.6 or greater
        ● WordPress Memory Limit: 64 MB or greater (128 MB or higher is preferred)

= Step-by-Step Installation Guide =
    1. Download the Plugin:
        ○ Obtain the WPelemento Importer plugin from the WordPress.org Plugin Repository or from WP Elemento's official website.
    2. Upload and Install the Plugin:
        ○ Using WordPress Admin Panel:
            1. Log in to your WordPress dashboard.
            2. Navigate to Plugins > Add New.
            3. In the search bar, type WPelemento Importer.
            4. Click Install Now on the WPelemento Importer plugin.
            5. After installation, click Activate to enable the plugin.
        ○ Manual Upload:
            1. Download the WPelemento Importer plugin zip file.
            2. Extract the zip file to obtain the plugin folder.
            3. Using an FTP client or your web host’s file manager, upload the extracted folder to the wp-content/plugins/ directory of your WordPress installation.
            4. Log in to your WordPress dashboard.
            5. Navigate to Plugins > Installed Plugins.
            6. Locate WPelemento Importer in the list and click Activate.

= Using the Plugin =
    1. Select a Theme or Template:
        ○ Choose a WP Elemento theme or template that you wish to use for your website.
    2. Install and Activate the Theme:
        ○ Go to Appearance > Themes in your WordPress dashboard.
        ○ Click Add New and upload your selected WP Elemento theme.
        ○ Install and activate the theme.
    3. Quick Start Setup:
        ○ After activating the theme, click on the Quick Start button provided by WPelemento Importer.
        ○ Follow the on-screen instructions to install all the required plugins necessary for the theme.
    4. Import Demo Content:
        ○ Click on the Import Demo button.
        ○ WPelemento Importer will automatically import all demo content, including images, settings, shortcodes, CSS, and more, replicating the theme demo exactly.
    5. Visit Your Site:
        ○ Once the import process is complete, visit your site to see it transformed into the demo layout you selected. Your site will now look and function exactly like the theme demo.

By following these steps, you can efficiently set up your WordPress website using WP Elemento themes with WPelemento Importer. This streamlined process ensures that even users with minimal technical expertise can achieve professional results quickly and easily
